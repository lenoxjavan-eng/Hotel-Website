# Anfield Star Hotel

A simple and responsive hotel website that allows users to explore rooms and make reservations online.

---

## Features

-  View available rooms with descriptions and prices  
-  Booking form for reservations  
-  Image gallery of the hotel  
-  Contact and location details  
-  Responsive design (works on mobile and desktop)  

---

## Technologies Used

- HTML5  
- CSS3  
- Bootstrap  

---

### Clone the repository

[git clone](https://github.com/Joshua-dev559/Hotel-Website.git)

### Run the website

Open `index.html` in your browser.

1. Go to the homepage  
2. Browse available rooms  
3. Go to the contact page  
4. Fill out your details  
5. Submit your inquiry  

---

##  Customization

You can edit:

- Hotel name in HTML files  
- Images in the `images/` folder  
- Styles in `css/styles.css`
